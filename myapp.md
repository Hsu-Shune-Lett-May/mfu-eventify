App Name:

1.1 One-sentence pitch:
Eventify is a mobile app that helps MFU university students keep track
of campus events and receive reminders so they don’t miss important
activities.

1.2 App category:
Productivity

2.1 Persona:
Age - 20 years old
Occupation - University student
Location - University campus
Tech Skill Level - Normal
Daily Problems - Forgets campus events, misses interesting activities,
overwhelmed by announcements from different platforms

2.2 Daily Problem:
The student often forgets or misses campus events because information
is scattered across different platforms and there is no simple
reminder system focused on campus activities.

3.1 Pain Point:
Campus events are announced through posters, social media, or chat
groups, which makes it easy for students to overlook dates and times.
Students have to remember events by themselves and often forget.

3.2 Why existing solutions not enough:
1. Social media posts get buried quickly and posts may not be seen
2. Calendar apps require manual input and are not campus-focused
3. Chat groups are noisy and hard to track specific events
There is no simple app focused only on campus events with reminders.

4.1 What the app does:
Eventify displays a list of campus events, allows students to view
details information, save events they are interested in, and sends
reminder notifications before the event starts.

4.2 How it solves the problem:
· Pain: Users forget event dates → App Action: Save events to personal
calendar → Result: Events are securely stored and accessible anytime
· Pain: Users miss events they wanted to attend → App Action: Set
reminders → Result: Users receive notifications before events start
· Pain: Users are overwhelmed by too many announcements → App Action:
Centralize all campus events in one app → Result: Reduced confusion
and easier event discovery

5. Core Features (MVP):
1. Event List - Shows upcoming campus events
2. Event Details - Displays date, time, location, description
3. Save Event List - Allows users to bookmark events
4. Reminder Notification - Sends alert before event
5. Create Event - Student can add an event

6. User Journey:
1.      User opens Eventify
2.      Views list of campus events
3.      Selects an event
4.      Saves event and sets reminder
5.      Receives notification before event starts
